YouTube Audio Downloader
========================

A Django web application for extracting the audio from a YouTube video and converts it to MP3 that a user can download.

**Main Site:** http://www.youtubeadl.com


Technology Stack
----------------
- Python 2.7
- Django 1.8
- Twitter Bootstrap 3
- PostgreSQL
- Nginx
- Gunicorn
- Celery
- Redis
- Supervisor
- Virtualenv
